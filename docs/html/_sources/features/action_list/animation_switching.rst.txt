Animation Switching
+++++++++++++++++++


- The Core Feature of the Addon is Animation Switching with Frame Range Adjusted when you switch the action

- This is Useful when you are working on multiple action sets, such as creating a animation set for your game character model

- The Action is stored on the Object itself, so it will only show actions that you have added to the object

- Animation Switching is activated when you click the Action in the list or Using the :ref:`Refresh Frame Range` Operator
